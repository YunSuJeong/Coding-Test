## 문제 풀이
로봇과 원점 사이의 최대 거리를 최대화하려면 L 또는 R이 가장 많을 때, 두 가지 경우만 비교하면 된다.  
L과 R이 번갈아 나오는 순간 원점과의 거리를 최대화 할 수 없기 때문이다. ex) LR = -1 + 1 = 0  
'?'를 모두 L 또는 R로 치환하여 최대 거리를 구한다.

## 문제 풀이
주어진 문자열이 재현이가 만들어 낼 수 있는 문자열인지 판단하는 문제

**[ 비교 조건 ]**
1. 문자열의 길이
2. 짝수인 경우, 같은 문자열이 두 번 반복되어 있는지
